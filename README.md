# homebrew-tap

Homebrew formulae for my CLI tools.

## Formulae

- **[ruth-cli](https://github.com/zhongdai/ruth-cli)** — agent-friendly TOTP authenticator CLI.
- **[bd-watcher](https://github.com/zhongdai/bd-watcher)** — lightweight TUI dashboard that watches [bd (beads)](https://github.com/steveyegge/beads) graph progress.

## Install

```bash
brew tap zhongdai/tap
brew install ruth-cli
brew install bd-watcher
```

## Install from HEAD (latest main)

```bash
brew install ruth-cli --HEAD
brew install bd-watcher --HEAD
```

## Update

```bash
brew upgrade ruth-cli
brew upgrade bd-watcher
```
